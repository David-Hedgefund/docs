Every scenario is built from a **theme**: a macro regime, sector mix, catalyst, and the analytical task it poses. This endpoint returns the full catalog so you can pick scenarios by theme.

## Scenario themes

```http
GET /metadata/scenario-themes
```

The full theme catalog: id, title, description, macro regime, sectors, market mechanics, and the agent task each theme poses. Takes no parameters.

<CodeGroup>
```bash cURL
curl -s "https://api.davidhf.com/metadata/scenario-themes" \
  -H "X-API-KEY: YOUR_API_KEY"
```

```python Python
dd.metadata.scenario_themes()
```
</CodeGroup>
